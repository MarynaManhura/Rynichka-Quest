# 🎈 Квест на 30 · Côte d'Azur

Веб-квест-сюрприз на день народження. Коханий відкриває посилання на телефоні і крок за кроком
знаходить заховані по квартирі подарунки. Дві перші доставки (кулі й торт) він фотографує —
і **фото миттєво приходить тобі в Telegram**.

Усе живе в **одному файлі** `index.html` — без сервера, без збірки. Просто відкрити в браузері
або задеплоїти перетягуванням.

---

## ⚙️ Крок 1. Налаштувати Telegram (2 хвилини, безкоштовно)

Щоб фото доставок приходили тобі в Telegram, потрібні два значення: **токен бота** і **твій chat id**.

### A. Створити бота й отримати `BOT_TOKEN`
1. У Telegram знайди **@BotFather** і відкрий чат.
2. Надішли команду `/newbot`.
3. Придумай назву бота, потім — унікальне ім'я (має закінчуватись на `bot`, напр. `marisha_quest_bot`).
4. BotFather пришле токен виду `123456789:AAdfk-Xy...` — це і є **BOT_TOKEN**.

### B. Дозволити боту писати тобі
1. Відкрий чат зі своїм новоствореним ботом (посилання дає BotFather).
2. Натисни **Start** / надішли йому будь-яке повідомлення.
   *(Без цього кроку бот не зможе тобі нічого надіслати.)*

### C. Дізнатися свій `CHAT_ID`
1. У Telegram знайди **@userinfobot** → натисни **Start**.
2. Він покаже твій `Id` — це число (напр. `987654321`). Це твій **CHAT_ID**.

---

## ✏️ Крок 2. Вставити ключі у файл

Відкрий `index.html` у будь-якому текстовому редакторі. На початку блоку `<script>` знайди:

```js
const BOT_TOKEN = "ВСТАВ-СЮДИ-ТОКЕН-БОТА";
const CHAT_ID   = "ВСТАВ-СЮДИ-СВІЙ-CHAT-ID";
```

Заміни на свої значення (лапки залиш):

```js
const BOT_TOKEN = "123456789:AAdfk-Xy...";
const CHAT_ID   = "987654321";
```

Збережи файл.

> 💡 **Перевірка:** відкрий `index.html` подвійним кліком, пройди перший крок, завантаж будь-яке
> фото й натисни «Надіслати Маріші». Якщо все правильно — фото з'явиться в чаті з ботом, а під
> кнопкою буде «✅ Фото надіслано».

### ✍️ Редагувати тексти кроків (за бажанням)
Нижче у тому ж файлі є масив `STEPS` — сім кроків. Кожен має:
- `type` — `'delivery'` (з фото) або `'find'` (пошук);
- `clue` — підказка, що бачить коханий;
- `gift` — назва подарунка;
- `note` — твоє тепле слово, що відкривається після знаходження.

Міняй тексти прямо в лапках — більше нічого чіпати не треба.

---

## 🚀 Крок 3. Задеплоїти (отримати посилання)

Вибери будь-який варіант — обидва безкоштовні й без реєстрації коду.

### Варіант 1 — Netlify Drop (рекомендовано)
1. Зайди на **https://app.netlify.com/drop**
2. **Перетягни файл `index.html`** у вікно браузера.
   *(Якщо є й README — краще перетягнути всю папку, але достатньо лише `index.html`.)*
3. Netlify одразу дасть посилання виду `https://назва.netlify.app` — це і є квест.
4. Скопіюй посилання й надішли коханому в день народження.

### Варіант 2 — tiiny.host
1. Зайди на **https://tiiny.host**
2. Завантаж `index.html` (або zip із ним).
3. Отримай посилання й поділись ним.

> 📱 Посилання відкривається з будь-якого месенджера прямо на телефоні. Прогрес зберігається —
> якщо він закриє вкладку, квест продовжиться з того ж кроку.

## ✅ Що працює
- 7 кроків з індикатором прогресу; екран привітання і романтичний фінал (квиток у Грецію).
- Доставки вимагають фото перед переходом далі; фото **стискається** на телефоні
  (до ~1280px, JPEG) і йде в Telegram. Якщо інтернет підвів — показує попередження,
  але дозволяє продовжити (не блокує).
- Прогрес зберігається в браузері (переживає перезавантаження).
- Мобіл-фьорст, адаптив до десктопа, стиль «Французька Рів'єра».
- Делікатний звук + вібрація при відкритті подарунка (вимикається кнопкою 🔊 у кутку).

## 🔒 Дрібниця про безпеку
Токен бота лежить у файлі (так влаштовані статичні сайти без сервера). Для одноразового
подарунка це нормально. Якщо схочеш — після свята можна видалити бота через @BotFather
(`/deletebot`), і токен стане недійсним.
